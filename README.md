# Pokemon Portal

## Requirements

    - Node 14+
    - yarn (recommended)

## Running

#### Install dependencies

    yarn

#### Scripts

- **Start dev server:** `yarn start`
- **Build:** `yarn build`
- **Test:** `yarn test`

## Stack

    React 18
    Typescript 4.5.5
    React-router 6
    Redux-toolkit
    Webpack 5
    MUI 5
